## 🚀 QUICK START GUIDE

### Prerequisites
- Java 17+ (JDK 17)
- Node.js 16+
- Maven 3.6+

### Option 1: Complete Backend Setup (Windows PowerShell)

```powershell
# Set Java Path
$env:Path = "C:\Program Files\Java\jdk-17\bin;$env:Path"

# Navigate to backend directory
cd C:\nathin\infosys\task\backend

# Clean, build, and run
mvn clean package -DskipTests -q
java -Xmx512m -jar target/shems-backend-1.0.0.jar
```

**Backend runs on:** http://localhost:3000/api

---

### Option 2: Frontend Setup (In a NEW Terminal)

```powershell
# Navigate to frontend directory
cd C:\nathin\infosys\task\frontend

# Install dependencies (first time only)
npm install

# Start development server
npm run dev
```

**Frontend runs on:** http://localhost:5173

## 🔌 API Endpoints (Backend)

### Authentication
- `POST /api/auth/register` - User registration
- `POST /api/auth/login` - User login
- `POST /api/auth/send-otp` - Send OTP to email
- `POST /api/auth/verify-email` - Verify email with OTP
- `POST /api/auth/forgot-password` - Request password reset
- `POST /api/auth/verify-reset-otp` - Reset password with OTP
- `POST /api/auth/logout` - Logout

### Devices
- `GET /api/devices` - Get all devices
- `POST /api/devices` - Add new device
- `PUT /api/devices/{id}` - Update device
- `DELETE /api/devices/{id}` - Delete device
- `GET /api/devices/{id}/consumption` - Get device consumption

### Dashboard
- `GET /api/dashboard/energy-stats` - Get energy statistics
- `GET /api/dashboard/today-consumption` - Get today's consumption
- `GET /api/dashboard/monthly-data` - Get monthly data
- `GET /api/dashboard/report/daily` - Download daily report
- `GET /api/dashboard/report/weekly` - Download weekly report
- `GET /api/dashboard/report/monthly` - Download monthly report

## 🔐 Authentication Flow

1. **Registration**: User → Email OTP → Verify Email → Complete Profile → Registered
2. **Login**: Email + Password → Check Email Verified → Generate Token → Access Dashboard
3. **Password Reset**: Email → Reset OTP → New Password → Password Reset Complete
4. **Session**: Token stored in LocalStorage → Auto-logout on token expiry

## 🐛 Troubleshooting

### Backend Won't Start
1. Ensure Java path is set: `java -version`
2. Check port 3000 is not in use: `netstat -ano | findstr ":3000"`
3. Delete old database files: `rm shems_db.*`
4. Rebuild: `mvn clean package -DskipTests`

### Frontend Won't Connect to Backend
1. Ensure backend is running on port 3000
2. Check CORS settings in backend (should allow localhost:5173)
3. Check browser console for specific error messages
4. Verify API base URL in `frontend/src/services/api.js`

### OTP Not Received
1. Email service requires proper Gmail SMTP credentials
2. For testing, check backend console logs for OTP values
3. Use the OTP shown in console to test verification
